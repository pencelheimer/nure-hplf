1. Реалізуйте кнопку, при натисканні на яку буде виводитися повідомлення
   "Вітаємо з першим додатком на Kotlin!".
2. Розробіть додаток для ведення списку завдань (ToDo). Користувач повинен мати
   можливість додавати та відмічати завдання виконаними.
3. Реалізуйте функціонал відправлення запиту до віддаленого сервера та
   отримання та відображення даних. Використайте Retrofit та Kotlin Coroutines.
4. Створіть додаток для зберігання та обміну рецептами. Кожен рецепт повинен
   включати назву, інгредієнти та інструкції приготування.
